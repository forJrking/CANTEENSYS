### 项目技术要点（初学者）

1. 使用原生数据库做用户信息存储
2. xml文件的数据解析搭配列表显示
3. 封装了数据库的CRUD
4. ViewPager的基本使用
5. 调用隐示意图启动打电话界面
6. Android的基础控件和四大组件的部分使用
